Анимация фоновых
объектов

Фоновые объекты — декоративные фигуры, плашки, которые появляются по мере прокрутки на сайте, чтобы сделать текст читаемым, либо усилить внимание пользователя на каком-то объекте. Анимация проигрывается по мере скролла, либо с помощью кнопок типа «Нажми на меня».



Например, пользователь кликает на круглую кнопку, весь фон становиться такого-же цвета и появляется черный текст. Это интерактивная анимация фонового объекта, предполагающая взаимодействие с интерфейсом.


Жми пример

Давайте рассмотрим пример


                        <div class="blok_yroka" id="les4">
                             <div class="ball" onclick="myFunction()">
                                Жми пример
                             </div>
                        </div>

Создадим класс для фонового блока и присвоим ему id

Внутри блока создадим еще один блок с классом 'ball'

И самое интересное

Пишем атрибут 'onclick' со значением "myFunction()" Это означает что при одном клике мышки по объекту вызывается и воспроизводиться функция 


                        .ball{
                            width: 200px;
                            height: 200px;
                            background-color:aqua;
                            border-radius: 100px;
                            text-align: center;
                            color: black;
                            line-height: 150px;
                            margin-inline: auto;
                            border:1px solid black;
                            
                          }

Зададим нашему кругу длину, высоту и цвет. 'border-radius' скругляет углы внешней границы элемента. Вы можете установить один радиус для создания круглых углов или два радиуса для создания эллиптических углов.


За установку горизонтального выравниванивания содержимого строкового уровня внутри элемента блока или поля ячейки таблицы отвечает свойство 'text-align', у него есть в основном 4 значения:


  • start(по левому краю)
  • end(по правому краю)
  • center(по центру)
  • justify(по ширине)

                        #les4{
                            transition-duration: 5s;
                          }
                          .ball:hover{
                            cursor: pointer; 
                            
                        }

Пропишем id в файлле CSS и придадим ему время анимации, также меняем курсор при наведении


И самое интересное...


                        <script>
                            function myFunction() {
                              if(document.getElementById("les4").style.backgroundColor != "aqua"){
                                document.getElementById("les4").style.backgroundColor = "aqua";
                                document.getElementById("les4").style.color = "black";}
                              else{
                                document.getElementById("les4").style.backgroundColor = "rgba(62, 56, 56, .9)";
                                document.getElementById("les4").style.color = "white";}  
                            }
                        </script>

Пишем тег <script> в конце html документа

И ставим условие: если в документе елемент с id "les4" не бирюзового цвета, то меняется на этот цвет и меняет цвет шрифта

Если такое условие не выполняется то при нажатии кнопки цвет будет меняться с бирюзового на темно-серый, а шрифт с черного на белый
