Анимация интерфейсов

Функциональная анимация интерфейсов касается инпутов, элементов навигации, секций, меню, появления уведомлений, pop-up окон, поведения блоков (например, FAQ — анимация по клику, открытие списков, закрытие)

меню




Сейчас мы создадим такое-же навигационное меню как указан в примере. 


                                <div class="lessons_2">
                                    <div class="nav">
                                        <div class="left_nav">logotip</div>
                                        <div class="right_nav">
                                            <div class="nav_button">Стать партнером</div>
                                            <div class="nav_button_anim"><img src="../img/phone_white.png">8-800-800-88-88</div>
                                            <div class="nav_button_anim"><img src="../img/email_white.png">misterfix @ gmail.com</div>
                                            <div class="nav_button">меню</div>
                                        </div>
                                    </div>
                                </div>

Создаем блок в котором у нас будет меню и назовем класс "lessons_2"

Зададим цвет фона и размеры


                        .lessons_2{
                            background-color: black ;
                            width: 98%;
                            height: 80px;
                        }

далее блоку с классом "nav" добавим парочку значений


                        .nav{
                            display: flex;
                            justify-content: space-between;
                        }

display Задает тип элемента и позволяет менять поведение элемента в потоке, а со значением flex мы превращаем блок в "flexbox" контейнер, который предоставляет инструменты для быстрого создания сложных, гибких макетов, и функции, которые были сложны в традиционных методах CSS.

При помощи justify-content мы выравниваем внутрилежашие блоки равномерно вдоль оси x


                        .left_nav{
                            line-height: 80px;
                        }
                        .right_nav{
                            display: flex;
                            gap: 20px;
                            margin-right: 20px
                        }

Создаем те самые внутри лежащие блоки, также правому блоку задаем gap в значение 20px

это означает что между блоками будет отступы в 20 пикселей



А теперь самое интересное - АНИМАЦИЯ


блоки с классом "nav_button" при наведении меняют цвет фона, разберем сначала их


                        .nav_button{
                            line-height: 80px;
                        }

line-height устанавливает интерлиньяж (межстрочный интервал) текста, отсчет ведется от базовой линии шрифта 

                        .nav_button:hover{
                            cursor: pointer; 
                            animation-duration: 1s;
                            animation-name: slideiin;
                            animation-fill-mode: forwards;
                        }
                        @keyframes slideiin {
                            from {
                            }
                            to {
                                background-color: orange;
                            }
                          }

:hover после названия класса указывает на то что эти свойства будут работать при наведении курсора мыши

сursor означает курсор мыши, а со значением pointer курсор изменится как-будто этот объект можно нажать и что-то произойдет

animation-name назначение имени анимации, у нас анимация называется "slideiin"

с помощью animation-duration указываем на длительность анимации

свойство animation-fill-mode определяет, какие стили должны применяться к элементу, когда анимация не проигрывается, в нашем случае forwards позволяет элементу по окончанию анимации применяется стиль последнего ключевого кадра

с помощью @keyframes вызываем анимацию и прописываем имя анимации, указываем какие свойства мы хотим изменить, в нашем случае цвет фона кнопки



Перейдем к блокам с классами "nav_button_anim" в которых при наведении выдвигается дополнительная информация


                        .nav_button_anim{
                            line-height: 80px;
                            height: 80px;
                            width: 50px;
                            overflow:hidden;
                        }
                        img{
                            height: 50px;
                         
                            vertical-align: middle;
                        }

Задаем межстрочный интервал, а также длину и ширину кнопки до увеличения

Свойство overflow определяет, необходимо ли для ограничения блока элемента обрезать, предусматривать полосы прокрутки или просто отобразить. И со значением hidden контент обрезается



Далее все так же как и ранее только здесь мы кроме цвета фона изменяем ширину блока


                        .nav_button_anim:hover{
                            cursor: pointer; 
                            animation-duration: 1s;
                            animation-name: slidein;
                            animation-iteration-count:1;
                            animation-fill-mode: forwards;
                        }
                        @keyframes slidein {
                            from {
                              width: 50px;
                            }
                            to {
                                background-color: orange;
                              width: 350px;
                            }
                          }
