Курсора мышки в этом веб-приложении


                        <div class="cursor"></div>
                        <div class="cursor2"></div>

Два класса 


                        .cursor{
                            position:fixed;
                            width: 50px;
                            height: 50px;
                            border: 1px solid white;
                            border-radius: 50%;
                            left: 0;
                            top: 0;
                            pointer-events:none;
                            transform: translate(-50%, -50%);
                            transition: .1s;
                          }

Задаем свойство 'position' со значением 'fixed' что позволяет фиксириовать курсор в одной точке

Настраиваем размер и границу


'pointer-events' устанавливает, при каких обстоятельствах (если таковые имеются) конкретный графический элемент может стать целью событий указателя

'transform' позволяет поворачивать, масштабировать, наклонять или сдвигать элемент


                        .cursor2{
                            position:fixed;
                            width: 8px;
                            height: 8px;
                            background-color: white;
                            border-radius: 50%;
                            left: 0;
                            top: 0;
                            pointer-events:none;
                            transform: translate(-50%, -50%);
                          }

Копируем свойства с предыдущего класса меняя следующие свойства:

  • размер уменьшаем
  • вместо рамки задаем фоновый цвет

                        <script>
                            var cursor = document.querySelector(".cursor");
                            var cursor2 = document.querySelector(".cursor2");
                            document.addEventListener("mousemove", function(e){
                                cursor.style.cssText = cursor2.style.cssText = "left: " + e.clientX + "px;"+" top: " + e.clientY + "px;";
                            });
                         </script>

С помощью javascript создаем 2 переменные которые присваиваем ранее прописанные классы

Также не без помощи javascript присваиваем нахождение мышки в ранее прописанные классы
