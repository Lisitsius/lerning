Анимация текста

Чаще это эстетическая анимация: текст каким-то нестандартным образом появляется на экране:



  • побуквенно (эффект печатной машинки),
  • из невидимости,
  • появление снизу вверх или слева направо,
  • с искажением или поворотом.


Используется для эстетики на сайте, передачи эмоций либо темпа повествования, а также создания акцента. Так контент будет интереснее воспринимать, внимание пользователя не будет рассеиваться на всём сайте, но сохранится на тексте. По мере прокрутки сайта, за счёт последовательного появления информации, глаза пользователя по этим акцентам будут идти ровно также — последовательно.


Эффект появления привлекает дополнительное внимание пользователя к тексту, заголовку, повышает дочитываемость.

Как такое сделать? Сейчас я вам покажу


                        <p class="text_anim" data-text="Эффект появления привлекает дополнительное внимание пользователя
                        к тексту, заголовку, повышает дочитываемость.">
                        Эффект появления привлекает дополнительное внимание пользователя к тексту, заголовку, повышает
                        дочитываемость.</p>

Начнем с тега <р> который используется для обозночения параграфа

Зададим класс и добавим атрибут 'data-text' копируя в него текст из параграфа


                        .text_anim{
                            position: relative;
                            color: black;
                          }

В классе прописываем свойтсво 'position' которая указывает, как элемент позиционируется в документе, а со значением 'relative' элемент позиционируется в соответствии с нормальным потоком документа, а затем смещается относительно себя на основе значений top, right, bottom и left

И цвет текста делаем черным


                        .text_anim::before{
                            content: attr(data-text);
                            position: absolute;
                            color: white;
                            width: 5px;
                            overflow: hidden;
                            white-space: nowrap;
                            border-right: 3px solid white;
                            animation: animate 12s linear infinite;
                          }

Создадим такой-же класс с псевдоклассом 'before', напишем свойство 'content' со значением 'attr(data-text)'

Прописываем свойтсво 'position' только теперь со значением 'absolute' Элемент удаляется из обычного потока документа, и для элемента в макете страницы не выделяется дополнительное пространство


Свойство white-space определяет, как обрабатывается пробелы внутри элемента и со значением nowrap cворачивает пробелы, но подавляет разрывы строк (перенос текста) внутри источника

Задаем размер и цвет для правой границы

Также создаем анимацию: придумываем имя, продолжительность анимации, и количество повтора(infinite-бесконечность)


                        @keyframes animate{
                            0%{
                                width: 0;
                            }
                          100%{
                            width: 100%;
                          }}

И завершаем написанием @keyframes
